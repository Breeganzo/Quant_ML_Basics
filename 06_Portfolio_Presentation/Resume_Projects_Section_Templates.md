# Resume Projects Section Templates

## Format Guidelines

- **Bullet points:** 2-3 per project
- **Action verbs:** Built, Developed, Implemented, Designed
- **Metrics:** Include numbers where possible
- **Tech stack:** List relevant technologies

## Templates by Project Type

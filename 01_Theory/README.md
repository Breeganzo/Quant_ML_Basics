## How to Use

1. **Read theory BEFORE starting daily notebooks**
2. **Focus on mathematical derivations** - understand WHY, not just HOW
3. **Take notes by hand** for key formulas (helps retention)
4. **Connect theory to finance applications** immediately
5. **Prepare interview talking points** as you read

## Content Standards

Each theory document includes:
- **Intuition**: Plain-English explanation
- **Mathematical formulation**: LaTeX derivations
- **Assumptions**: When the model works/fails
- **Finance application**: Real use cases
- **Production considerations**: Latency, costs, explainability
- **Interview questions**: What firms ask about this topic
- **Further reading**: Papers and resources

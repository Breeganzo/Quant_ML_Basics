# Probability and Expected Value Questions

This folder contains probability problems commonly asked at quant interviews.

## Categories

1. **Dice and Cards** - Basic probability
2. **Expected Value** - Calculation and optimization
3. **Conditional Probability** - Bayes' theorem applications
4. **Games and Betting** - Strategic probability
5. **Martingales** - Stopping time problems

## Difficulty Levels

| Level | Description | Example Firms |
|-------|-------------|---------------|
| Easy | Basic calculations | All firms |
| Medium | Multi-step reasoning | All firms |
| Hard | Complex optimization | Jane Street, HRT |

## Practice Strategy

1. Start with easy problems to build confidence
2. Time yourself (5-10 minutes per problem)
3. Practice mental math
4. Explain your reasoning out loud

## Sample Problems

### Easy
1. What's the expected value of rolling a fair die?
2. Probability of drawing a heart from a standard deck?

### Medium
1. Expected value of the maximum of two dice rolls?
2. Probability of getting exactly 3 heads in 5 flips?

### Hard
1. Two dice rolled until same number appears. Expected rolls?
2. Optimal stopping problem with sequential observations
